# 文档样本目录

本目录用于存放测试用的文档样本文件。这些文件仅用于测试目的。

## 文件说明

- `sample.pdf`: PDF 文档样本
  - 文件类型: PDF
  - 大小: 2048 字节
  - 用途: 用于测试 ExifTool 分析器的 PDF 文档分析功能

## 使用说明

1. 在测试代码中使用 `get_sample_path("pdf")` 获取样本文件路径
2. 使用 `get_sample_info("pdf")` 获取样本文件的完整信息
3. 确保在测试完成后清理所有临时文件

## 维护说明

1. 定期检查样本文件的有效性
2. 更新样本文件时同步更新测试用例
3. 记录样本文件的特征和用途
